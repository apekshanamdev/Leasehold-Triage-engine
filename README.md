# Leasehold-Triage-engine
AI-assisted workflow system for leasehold arrears triage and credit control checks.

## Live Prototype

[View Prototype Here](silly-alfajores-0d7eca.netlify.app)

## Overview
A working tool that runs a portfolio of accounts through a ten-gate decision engine grounded in UK credit control rules, and tells you exactly what to do with each one.

Built as an experiment around operational automation and real estate credit control operations.
